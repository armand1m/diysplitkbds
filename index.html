Essential skills and tools

  • Basic soldering skills: for getting the pieces together
  • Some electronics skills: for choosing the right components and debugging why your LED is not working
  • Some coding knowledge: for firwmare customization and flashing
  • Curiosity and Patience. A lot of it. You'll mess up.

Main components

  • A keyboard case
  • Switches and keycaps
  • Microcontroller
  • Diodes (depending on the number of keys)

Cantor: a diodeless 42 key keyboard

Github: https://github.com/diepala/cantor

Popular opensource firmware

  • QMK: docs.qmk.fm
  • ZMK, popular for wireless keyboards: zmk.dev
  • Your not built yet custom written firmware in Rust because you're curious
  • VIAL, for those who don't like to code

Resources for beginners

  • Build guides from splitkb.nl
  • QMK Firmware Docs: https://docs.qmk.fm/
  • Compare different keyboard layouts: https://compare.splitkb.com/
